The $30/month CHRO in a Markdown File

There's a newsletter selling a "CHRO skill" for Claude right now.

Install this file, it says, and ask it the questions you can't ask anyone at work. It responds like a seasoned Fortune 100 CHRO. Paid subscribers only.

I read the pitch so you don't have to. It's the same play as the "50 HR skills, comment COMPLETE below" posts flooding this feed.

And here's the annoying part.

The underlying idea is actually good.

A well-built skill file really can give an AI the operating posture of a seasoned CHRO. The pushback on bad framing. The questions that cut through. The board-prep structure, the layoff sequencing, the comp committee instincts. That's real, and it's useful at 11pm on a Sunday when the comp chair has a bright idea and counsel isn't picking up.

What's not real is the rest of the pitch.

A markdown file does not contain Radford or Mercer data. Those are licensed products. Any file claiming otherwise is making up numbers.

A markdown file does not know current employment law. WARN thresholds, AI hiring rules, works council obligations. These change constantly, and a static file froze the day it was written.

A markdown file is not thirty years of judgment. It's a well-briefed thought partner. Useful. Different thing.

So I built the honest version. And it's free.

The CHRO Thought Partner. One file. It carries the posture, the mental models, the playbook skeletons, the signature questions, and five worked examples of the decisions nobody trains you for.

It also does something the paywalled versions won't.

It tells you where it ends.

It runs hard decisions through five lenses (strategic, tactical, labor, technology, comp) and tells you where they disagree. Because the actual CHRO job isn't getting an answer. It's adjudicating between the GC, the CFO, and the works council when they're all correct.

It refuses to invent market comp numbers. It says so.

Every playbook that touches law carries an explicit gate: verify current status, then call counsel for anything with real exposure. In that order.


And the last section is called "What this skill cannot be." Judgment without data access is a thought partner. Judgment with governed access to real payroll, benefits, and talent data, with an audit trail a regulator can inspect, is a platform. No markdown file closes that gap, and any pitch implying otherwise is selling you the menu and calling it dinner.

Take the file. Fork it. Argue with it. It's yours.

The thinking is free. It always should have been.
